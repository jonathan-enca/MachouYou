# Machouyou — Dashboard Funnel Meta Ads

Dashboard de suivi des performances média buying Machouyou.

## Déploiement

1. Push ce repo sur GitHub
2. Connecte-le à Vercel (il détecte Next.js automatiquement)
3. C'est en ligne 🚀

## Développement local

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000)

## Mise à jour des données

Remplace le contenu de `RAW_DATA` dans `components/Dashboard.jsx` avec les nouvelles données exportées depuis Pipeboard MCP.
